Unit Testing & Code Coverage

Unit Testing

All work is ran through a unit testing process.

KarmaJS tests should be ran & pass locally before commiting code to Stash.

Enzyme is used rather than React Test Utilities for a better test API.

The KarmaJS tests are also ran on Bamboo when merging work into the development branch.

# Run test with PhantomJS
$ npm test

# Run test with Chrome
$ npm run test:watch

Code Coverage

All components are ran through isparta for code coverage reporting. All components should have 90% or higher code coverage.

To see the code coverage report run the following:

$ npm run coverage
